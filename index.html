How To Use

Step 1: Include relative javascript files

<script type="text/javascript" src="jquery.js"></script>
        <script type="text/javascript" src="jQpaginate.js"></script>

Step 2 (optional): Include stylesheet with both themes

<link rel="stylesheet" href="styles.css" media="screen"/>

Step 3: call your function

        $(function() {
        	$('table#myTable').paginate();
        });

Available Options

items Integer Default: 1 Total number of items that will be used to calculate the pages.
perPage Integer Default: 10 Number of items displayed on each page.
displayedPages Integer Default: 2 How many page numbers should be visible while navigating.
Minimum allowed: 2 (previous, current & next)
currentPage Integer Default: 1 Which page will be selected immediately after init.
hrefTextPrefix String Default: "#page-" A string used to build the href attribute, added before the page number.
hrefTextSuffix String Default: empty string Another string used to build the href attribute, added after the page number.
theme String Default: "simple" The class of the CSS theme.
onInit Function Optional Function to call when the pagination is initialized.
